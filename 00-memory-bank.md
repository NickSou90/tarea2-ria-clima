# Contexto del Proyecto: Tarea 2 RIA - Clima en Tiempo Real

## Objetivo
Desarrollar una aplicación RIA que consuma la API de OpenWeather, con al menos 2 rutas, usando React y Material Design.

## Stack Tecnológico
- Framework: React + Vite
- UI: Material-UI (MUI)
- Rutas: React Router
- Gestor de paquetes: pnpm

## Reglas
- No usar base de datos (solo LocalStorage).
- No incluir lógica de negocio en el backend.
